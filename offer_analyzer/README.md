# Chinese Tech Job Offer Analyzer

A brutally honest command-line tool to compare job offers in the Chinese tech industry. Built as a portfolio project to demonstrate C++ fundamentals.

## 🚀 Features
- View multiple job offers with salary, housing stipend, 996 culture, and stock options.
- Filter offers to avoid 996 companies.
- Find the best financial offer.
- Find the best work-life balance offer.
- Get a final, brutally honest recommendation.

## 🛠️ Tech Stack
- C++17
- Standard Template Library (STL): `vector`, `algorithm`, `iostream`
- Compiled with `g++` on Ubuntu (WSL2)

## 📦 Installation
```bash
git clone https://github.com/Yaounde-CS-Student/cpp-grind.git
cd cpp-grind/day7_offer_analyzer
g++ -std=c++17 -o offer_analyzer offer_analyzer.cpp
./offer_analyzer
